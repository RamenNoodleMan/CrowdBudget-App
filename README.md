# 🛒 CrowdBudget

**Community-Based Shopping Budget Estimation Using Crowdsourced Price Data**

---

## 📱 Overview

CrowdBudget is a mobile application that helps users estimate their shopping budget using crowdsourced price data. Users can view product prices contributed by other users, add items to a shopping basket, submit prices they've paid, and see real-time price averages.

---

## ✨ Features

- **User Authentication** - Register and login securely
- **Product Search** - Search for products with real-time filtering
- **Category Filters** - Filter by Bakery, Dairy, Meat, Vegetables, and more
- **Shopping Basket** - Add items with quantity selection
- **Crowdsourced Pricing** - Submit prices you've paid for products
- **Real-Time Price Averaging** - See average prices from community contributions
- **Price Trends** - View 📈/📉 trends based on historical data
- **Basket History** - Save and view previous shopping lists
- **Total Budget Calculation** - See your estimated total in ZMW

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **Platform:** Android (Minimum SDK 24)
- **Backend:** Firebase Firestore
- **Authentication:** Firebase Authentication
- **Currency:** ZMW (Zambian Kwacha)
